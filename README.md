# Catalog Spring Boot App

A product catalog management system built with Spring Boot that provides RESTful APIs for managing products in a catalog.

## About This Project

This project is a full-stack catalog application developed using Spring Boot framework. It implements a complete CRUD (Create, Read, Update, Delete) system for managing product information including product details like name, description, price, and category.

## What I Built

- **RESTful API Service**: Created REST endpoints to handle product operations
- **Product Management System**: Implemented functionality to add, view, update, and delete products from the catalog
- **Database Integration**: Connected the application with a database using Spring Data JPA
- **Controller Layer**: Developed controllers to handle HTTP requests and responses
- **Service Layer**: Implemented business logic for product operations
- **Repository Layer**: Created data access layer for database operations
- **Entity Models**: Designed product entity models with proper annotations

## Technologies & Tools Used

### Backend Framework
- **Spring Boot** - Main application framework
- **Spring Web** - For building REST APIs
- **Spring Data JPA** - For database operations and ORM
- **Hibernate** - ORM framework for database mapping

### Language
- **Java** - Programming language

### Build Tool
- **Maven** - Dependency management and build automation

### Database
- **H2 Database** - In-memory database for development and testing
- **MySQL/PostgreSQL** - Production database options

### Development Tools
- **Spring Tool Suite (STS)** / **IntelliJ IDEA** / **Eclipse** - IDE
- **Postman** - API testing
- **Maven Wrapper** - Included for running Maven without installation

## Key Features Implemented

1. **Get All Products** - Retrieve list of all products in the catalog
2. **Get Product by ID** - Fetch details of a specific product
3. **Create Product** - Add new products to the catalog
4. **Update Product** - Modify existing product information
5. **Delete Product** - Remove products from the catalog

## Project Architecture

The application follows a layered architecture:

- **Controller Layer** - Handles HTTP requests and responses
- **Service Layer** - Contains business logic
- **Repository Layer** - Manages data persistence
- **Model/Entity Layer** - Defines data structures
